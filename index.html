About the project



OSCalc: An Open Source Java Spreadsheet Application OSCalc is a Java spreadsheet project I developed for a university assignment. It showcases my object-oriented programming skills and Java knowledge. With OSCalc, you can work with multiple sheets, cross-sheet references, complex equations, and easily create custom functions. It can save and load data in JSON and CSV formats, using my custom JSON parsing library. It also includes undo/redo functionality, enhancing usability. This project demonstrates my Java and OOP skills as a university student.
